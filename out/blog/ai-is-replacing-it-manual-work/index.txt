<h2>What I Built — And What It Replaced</h2>
<h3>1. SNOW SLA Monitor (Python Daemon)</h3>
<p>The first thing I automated was ServiceNow SLA chasing. I was manually checking tickets every hour, worried about breaches. Built a Python script that runs every 5 minutes via cron:</p>
<pre><code class="language-python"># Polls SNOW REST API, calculates breach time, fires WhatsApp 30 min before breach
import requests, schedule
from datetime import datetime

def check_sla_breaches():
    tickets = get_snow_tickets()
    for ticket in tickets:
        mins_remaining = calculate_sla_remaining(ticket)
        if mins_remaining &#x3C;= 30:
            send_whatsapp_alert(ticket, mins_remaining)

schedule.every(5).minutes.do(check_sla_breaches)
</code></pre>
<p><strong>Result:</strong> Zero SLA breaches for 6 months. Saved 45 min/day of manual monitoring.</p>
<h3>2. Daily Email Summary Agent</h3>
<p>Every morning, a Python script reads the corporate Outlook inbox via Microsoft Graph API, groups emails by priority, and produces a clean action list:</p>
<blockquote>
<p>"3 tickets assigned · 1 vendor quote needs approval · 2 meeting invites · 1 urgent from senior leadership"</p>
</blockquote>
<p><strong>Result:</strong> Email triage time dropped from 40 minutes to 5 minutes per day.</p>
<h3>3. IT Asset Manager (Web App)</h3>
<p>Replaced an Excel file with 1,500+ rows that three people were editing simultaneously (and breaking). Built a Flask web app with proper search, filters, audit logs, and AI chat.</p>
<p><strong>Result:</strong> Asset query time from 10 minutes (Excel search) to 10 seconds (AI chat: <em>"show me all MacBooks assigned to the branch office under warranty"</em>).</p>

<h2>The Honest Reality Check</h2>
<p>AI didn't replace my job. It replaced the <em>worst parts</em> of my job — the repetitive, low-value administrative tasks that were draining my energy and preventing deep work.</p>
<p>The 100 minutes I recovered every day? I used them to:</p>
<ul>
<li>Learn Python ML (leading to the SNOW SLA Predictor project)</li>
<li>Build better relationships with end users (instead of rushing them off calls)</li>
<li>Propose a PowerBI dashboard to IT leadership (which got approved)</li>
<li>Study for the Azure Security certification</li>
</ul>
<blockquote>
<p><strong>The engineers who thrive in the next decade won't be the ones who resist AI. They'll be the ones who use it to amplify what humans do best — judgment, relationships, creativity, and strategic thinking.</strong></p>
</blockquote>
<h2>Where to Start</h2>
<p>If you're in corporate IT and want to start automating, here's the priority order:</p>
<ol>
<li><strong>Email summary</strong> — highest immediate ROI, requires only Microsoft Graph API access</li>
<li><strong>SNOW/ITSM alerts</strong> — REST API available in every modern ITSM tool</li>
<li><strong>Asset reporting</strong> — replace your Excel with even a basic SQLite + Python web app</li>
<li><strong>Compliance checks</strong> — scheduled PowerShell or Python scripts save hours of manual auditing</li>
</ol>
<p>Start small. One script. One problem. The compounding effect is real.</p>
